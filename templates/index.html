Konfirmasi Hapus Data
Apakah Anda yakin ingin menghapus semua input yang telah Anda isi? Data yang dihapus tidak dapat dikembalikan lagi, dan lembar kerja akan kembali kosong dari awal.
Pengaturan AI (OpenRouter)

Masukkan API Key OpenRouter untuk mengaktifkan fitur Asisten AI. API Key hanya disimpan di browser Anda (Local Storage).

Pilih Auto agar sistem otomatis pilih model gratis terbaik. Atau klik Muat dari OR untuk daftar lengkap real-time. API Key: openrouter.ai/keys
Template Laporan Cepat

Pilih template di bawah ini untuk menambahkan struktur standar secara otomatis ke Laporan Anda.

Jika BAB sudah ada, template akan melanjutkan ke BAB tersebut dan hanya menambahkan sub-bab yang belum ada.
